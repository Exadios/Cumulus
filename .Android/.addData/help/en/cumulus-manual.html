Manual Navigation

If you don't have a GPS connected to Cumulus, or the GPS can't get a good fix, Cumulus switches to Manual Navigation Mode. In this mode, the glider symbol on the map is replaced by an X symbol, and the glider's altitude is fixed at 1000 m. You can now use the cursor to move this X around on the map. The selected position is used for all calculations and is displayed in the status bar.

You now have the following keys available:

  • Arrows - move across the map (up is north)
  • C - centre on currently selected waypoint/airfield
  • SHIFT+H - centre on home site
