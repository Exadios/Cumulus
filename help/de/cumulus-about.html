Über Cumulus

Cumulus ist ein Flug Navigations Programm, das in erster Linie dem Segelflug dient. Es wurde von KFLog, dem KDE Flugauswertungsprogramm abgeleitet. Mit KFLog bildet es eine Familie, kann aber auch vollkommen eigenständig eingesetzt werden.

Cumulus wird als Open Source unter der GPL Lizenz veröffentlicht.

Cumulus wurde im Zeitraum von 2002-2008 entwickelt von:

  • André Somers
  • Eggert Ehmke
  • Axel Pauli (axel@kflog.org)
  • Eckhard Völlm
  • Hendrik Müller
  • Josua Dietze
  • Michael Enke
  • Florian Ehinger
  • Heiner Lamprecht
Es hat seine Wurzeln in KFLog, welches hier http://www.kflog.org zu finden ist.

Bitte wenden Sie sich bei Problemen an mich, Axel Pauli, Emailadresse siehe oben oder an unsere Mailingliste und fügen Sie eine vollständige Beschreibung der Version bei, die Sie benutzen. Drücken Sie die Tastenkombination SHIFT-V um die Version zu erfahren und beschreiben Sie, welche Schritte zu dem Problem führten. Bitte beschreiben Sie auch, welche Karten auf Ihrem PDA installiert sind und an welcher Position Sie sich aufhielten.

Die letzte offizielle Version finden Sie stets unter http://cumulus.kflog.org im Download Bereich.
Die neusten Programmquellen können mittels subversion von der folgenden URL https://svn.kflog.org/svn/repos/cumulus/qt4 runter geladen werden.

Letzte Aktualisierung: 27.05.2008 von Axel Pauli
