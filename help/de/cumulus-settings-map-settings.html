Einstellungen

Karte

Hier können Sie einige Einstellungen für die Darstellung der Karte festlegen.

Wenn Sie Probleme mit dem Speicherverbrauch haben oder wenn Sie die Menge der dargestellten Information reduzieren möchten, können Sie einige optionale Elemente auf der Karte deaktivieren. Diese Elemente werden dann nicht mehr geladen. Danach müssen Sie aber Cumulus beenden und neu starten, damit diese Optionen wirksam werden.

Das Deaktivieren von einzelnen Kartenelementen beschleunigt generell das Laden der Karten.

Das Laden der Höhenlinien kann ebenfalls deaktiviert werden. Wenn Höhenlinien aktiviert sind, wird die Karte entsprechend der Höhe eingefärbt. Diese Einfärbung ist detailierter, wenn Sie einen hohen Zoom Faktor wählen, im anderen Fall wird sie grober. Beachten Sie, dass die Höhenlinien für andere Funktionen die die Bodenhöhe benötigen essenziell ist.

Das Ausschalten der Darstellung von Wegpunkten löscht diese auf der gesamten Kartendarstellung. Durch drücken der Taste A kann man zwischen beiden Möglichkeiten hin- und herschalten.

Die Kurslinie ist eine Linie von Ihrer gegenwärtigen Position zum ausgewählten Zielpunkt. Sie wird als lila Linie gezeichnet.

Im Menü Weitere Optionen..., in welches man durch drücken des entsprechenden Knopfes gelangt, lassen sind weitere Kartenspezifika einstellen.
