Vorflug Einstellungen

Aufgabe

Dieses Fenster ist in zwei Subfenster unterteilt. Im oberen Fenster sind alle von Ihnen definierten Aufgaben zu sehen. Klicken Sie eine Aufgabe an, so werden alle Details von ihr im unteren Fenster angezeigt. Gleichzeitig wird diese Aufgabe als die zu fliegende angesehen, wenn Sie das Menü über die OK Taste verlassen. In diesem Fall wird die ausgewählte Aufgabe auf der Karte gezeichnet. Um die automatische Weiterschaltung beim Erreichen eines Wendepunktes zu aktivieren, muss man in der Aufgabenanzeige, den ersten Anflugpunkt per Hand selektieren. Wenn Sie den ersten Anflugpunkt bereits am Boden selektieren, wird die automatische Weiterschaltung erst aktiviert, wenn Sie sich schneller als 35 Km/h bewegen.

Hat man sich dem Ziel bis auf eine bestimmte Entfernung genähert, die man über die Vorflugeinstellungen-Allgemein unter Aufgabenpunkt Umschalteradius einstellen kann, wird automatisch zum nächsten Aufgabenpunkt weitergeschaltet.

Eine bestehende Aufgabe kann man durch das Klicken auf die entsprechnden Ikonen in der oberen rechten Fensterecke löschen bzw. verändern. Wenn Sie eine neue Aufgabe definieren wollen, so klicken Sie die Editor Ikone an. Damit wird ein neuer Bildschirm geöffnet, in dem Sie Wegpunkte und Flugplätze aus einer umschaltbaren Liste auswählen können, die zu Ihrer Aufgabe benutzt werden sollen. Benutzen Sie die Pfeile zum hinzufügen bzw. löschen von Wegpunkten oder zum abändern der Reihenfolge Ihrer Wegpunkte in der Aufgabe.
