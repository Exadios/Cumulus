Vorflug Einstellungen

Aufgabe

Hier haben Sie die Möglichkeit, eine Flugaufgabe zu definieren. Das Fenster ist in zwei Subfenster unterteilt. Im oberen Fenster sind alle von Ihnen definierten Aufgaben zu sehen. Klicken Sie eine Aufgabe an, so werden alle Details von ihr im unteren Fenster angezeigt. Gleichzeitig wird diese Aufgabe als die zu fliegende angesehen, wenn Sie das Menü über die Ok Taste verlassen. In diesem Fall wird die ausgewählte Aufgabe auf der Karte gezeichnet.

Cumlus merkt sich die zuletzt von Ihnen selektierte Aufgabe. Wenn Sie Ihre Auswahl rücksetzen wollen, so klicken Sie den obersten Listeneintag an und verlassen das Fenster mit der Ok Taste.

Im oberen Teil des Fensters sind drei Spinboxen zu sehen. In der linken Spinbox können Sie Ihre geplante wahre Fluggeschwindigkeit durch die Luft (TAS) vorgeben. Daneben können Sie noch die Windrichtung (WD) und die Windgeschwindigkeit (WS) eingeben. Wenn die TAS und die Windgeschwindigkeit größer Null sind, werden diese Werte in der Aufgabenberechnung mit berücksichtigt. In diesem Fall sehen Sie im unteren Fenster unter der Spalte TH (true heading) den Kurs den Sie fliegen müssen, um Ihr Ziel unter Windeinfluss zu erreichen. Im anderen Fall ist die Spalte mit TC (true course) beschriftet, weil die TAS bzw. die Windgeschwindigkeit Null sind. Wenn die Winddreiecksberechnung fehl schlägt, wird die Spalte ebenfalls mit TC (true course) beschriftet. In diesem Fall werden die Spalten Windkorrekturwinkel (WCA) und Geschwindigkeit über Grund (GS) mit einem Minuszeichen belegt.

Um die automatische Weiterschaltung beim Erreichen eines Wendepunktes zu aktivieren, muss man in der Aufgabenanzeige, den ersten Anflugpunkt per Hand selektieren. Alternativ man man das auch über die Taste b machen. Aber beachten Sie folgenden Unterschied. Die Taste b wählt immer den ersten Anflugpunkt der Aufgabe aus, während Sie in der Aufgabenanzeige einen beliebigen Anflugpunkt auswählen können.

Wenn Sie den ersten Anflugpunkt bereits am Boden selektieren, wird die automatische Weiterschaltung erst aktiviert, wenn Sie sich schneller als 35 Km/h bewegen.

Haben Sie sich dem Ziel bis auf eine bestimmte Entfernung genähert, wird automatisch zum nächsten Aufgabenpunkt weitergeschaltet. Die entsprechenden Entfernungen stellen Sie im Menüpunkt Einstellungen/Aufgaben ein. Fliegen Sie an einem Aufgabenpunkt zu weit vorbei, wird nicht automatisch auf den nächsten Aufgabenpunkt weiter geschaltet. Das müssen Sie dann in der Aufgabenanzeige selber von Hand tun.

Eine bestehende Aufgabe können Sie durch das Klicken auf die entsprechnden Ikonen in der oberen rechten Fensterecke löschen bzw. verändern. Wenn Sie eine neue Aufgabe definieren wollen, so klicken Sie die Editor Ikone an. Damit wird ein neues Fenster geöffnet, in dem Sie Wegpunkte, Flugplätze und Landefelder aus einer umschaltbaren Liste auswählen können, die zu Ihrer Aufgabe benutzt werden sollen. Benutzen Sie die Pfeile zum hinzufügen bzw. löschen von Wegpunkten oder zum Abändern der Reihenfolge Ihrer Wegpunkte in der Aufgabe.
