Segler

Hier wählen Sie das Flugzeug aus, welches Sie benutzen wollen.

Flugzeug Auswahlfenster
In diesem Fenster können Sie eines der Flugzeuge auswählen, die Sie zuvor bei Einstellungen/Flugzeuge definiert haben. Der Flieger wird mit Typ, Kennzeichen und Wettbewerbskennzeichen hier angezeigt.
Co-Pilot
Name des Co-Piloten, wenn der Flieger ein Doppelsitzer ist.
Zuladung
Hier können Sie die Zuladung, wie das Gewicht des Piloten, Gepäck, usw. eingeben.
Wasser Ballast
Hier können Sie den Wasserballast angeben, den Sie vor dem Start eingefüllt haben.
