Settings

Press SHIFT+S to open directly the settings screen. The other way is, to press the menu button or the shortcut key m on the PDA to open the menu bar. Then navigate to Setup/General.

This dialog contains all basic configuration of Cumulus. It is separated into 12 sections:

  • Personal
  • GPS
  • Gliders
  • Maps
  • Map Objects
  • Terrain Colors
  • Task
  • Airfields
  • Airspaces
  • Units
  • Information
  • Look&Feel

Note! If you have done modifications in the single setting tabulators and you want to make them permanent you must leave this window by pressing the OK button. To cancel all done modifications, use the CANCEL button instead of.

FYI, all internal configuration data are stored by Cumulus in the user directory $HOME/.config in the file Cumulus.conf. Do only modifications there, if you know what you do and Cumulus is not running!

WARNING: Make your settings while safely on the ground, not while flying!
