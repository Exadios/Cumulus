Settings

Press SHIFT+S to open the settings screen. To open the menu, press the menu button on the PDA.

The settings dialog contains all basic configuration of Cumulus. It is separated into ten sections:

Personal
Gliders
Sector
GPS
Map Settings
Map Objects
Airfields
Airspaces
Units
Information

FYI, all internal setup data are stored by Cumulus in the user directory $HOME/.config in the file Cumulus.conf. Do only modifications there, if you know what you do.

WARNING: Make your settings while safely on the ground, not while flying!
