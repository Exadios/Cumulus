openAIP - Navigation Data

openAIP has the goal to deliver free, current and precise navigational data to anyone. openAIP is a web-based crowd-sourced aeronautical information platform that allows registered users to add, edit, download and comment aeronautical data in many common formats used in General Aviation.

As first you should register youself by the openAIP project, that you can use their provided data.

Please read and accept the Terms and Conditions from openAIP, before you download and use their data with Cumulus.

Caution! Please note, that all openAIP files carry the extension .aip. You should never change that extension otherwise Cumulus cannot recognize these files.

openAIP Airfield and Navaid Data

Cumulus can download and install on user request openAIP airfield and navaid files. Go to the Settings/Point Data - openAIP page and make there your desired settings.

openAIP Airspace Data

Cumulus can download and install on user request openAIP airspace files. Go to the Settings/Airspaces page and make there your desired settings.

openAIP Point Data

Cumulus can use thermal hotspot point data from openAIP. You can download such data from the webpage of the openAIP project, one file per country and import these data in to Cumulus via the menu point Preflight Settings - Waypoint Import.

Please help to improve the openAIP data. If you found mistakes, correct them. If you miss something, add it to the openAIP data base.
