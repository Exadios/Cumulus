Maps - openAIP

For airfields Cumulus can use the openAIP data base as alternative to Welt2000. Such data are provided by www.openaip.net at their webpage, one file per country.

openAIP has the goal to deliver free, current and precise navigational data to anyone. openAIP is a web-based crowd-sourced aeronautical information platform that allows registered users to add, edit and download aeronautical data in many common formats used in general aviation.

Please read and accept the Terms and Conditions from openAIP, before you download and use their data with Cumulus.

Cumulus can download automatically openAIP airfield files. Go to the Settings/Airfields - OpenAIP page and enter there the countries to be downloaded. If the Download button is pressed, the desired countries are fetched from the Internet and installed in the directory .../maps/airfields. The files carry the file extension .aip. To limit the amount of read airfield data, Cumulus provides two ways on the configuration page Settings/Airfields - OpenAIP:

  1. Filtering by countries. Press the Load button to select the files to be loaded. One file contains one country.
  2. Filtering by a radius around your home position.

Please help to improve the openAIP data. If you found mistakes, correct them. If you miss something, add it to the openAIP data base.
