openAIP - Navigation Data

openAIP has the goal to deliver free, current and precise navigational data to anyone. openAIP is a web-based crowd-sourced aeronautical information platform that allows registered users to add, edit, download and comment aeronautical data in many common formats used in General Aviation.

As first you should register youself by the openAIP project, that you can use their provided data.

Please read and accept the Terms and Conditions from openAIP, before you download and use their data with Cumulus.

Caution! Please note, that all openAIP files carry the extension .api. You should never change that extension otherwise Cumulus cannot recognize these files.

openAIP Airfield Data

Cumulus can download automatically openAIP airfield files. Go to the Settings - Airfields page, select as source OpenAIP and enter there the countries to be downloaded. If the Download button is pressed, the desired countries are fetched from the Internet and installed in the directory .../maps/airfields. The files carry the file extension .aip. To limit the amount of read airfield data, Cumulus provides two ways on the configuration page Settings - Airfields - OpenAIP:

  1. Filtering by countries. Press the Load button to select the files to be loaded. One file contains one country.
  2. Filtering by a radius around your home position.

openAIP Airspace Data

Cumulus can load openAIP airspace data. You can find such data on the webpage of the openAIP project, one file per country. You have to download by hand your desired countries. The downloaded files have to be installed in the directory .../maps/airspaces of Cumulus. Further help you will get on page Maps - Airspace.

openAIP Point Data

Cumulus can load navigational aid and thermal hotspot point data from openAIP. You can find such data on the webpage of the openAIP project, one file per country. Cumulus can import these data via the menu Preflight Settings - Waypoint Import.

Please help to improve the openAIP data. If you found mistakes, correct them. If you miss something, add it to the openAIP data base.
