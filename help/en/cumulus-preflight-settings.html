Preflight Settings

Press p to open the pre-flight settings screen. The same is possible by using the menu item Setup/Pre-flight.

The pre-flight settings dialog contains all the parameters you need to set before becoming airborne.

It is separated into three sections:

  1. Glider
  2. Task
  3. Common

WARNING: Make your settings while safely on the ground, not while flying!
