Settings

Sector

Here you can define the task point schemes and the drawing opitions. The task point scheme defines how the automatic switch to the next task point is executed, if a flight task is active. A flight task can be defined and activated in the Setup Preflight Task Dialog. The drawing options define the look of the task points.

TP Scheme
Two task point schemes are supported, Cylinder and Sector. The activated check box will make available the appropriate definition area.
Switch Scheme
The automatic task point switch provides two modes. Nearst mean the switch to the next task point will be executed if the minimum distance to the current task point was reached. Touched means the outer border of the cylinder or sector has been arrived. Furthermore in case of arrive the IGC logger interval will be set automatically to one second for a duration of 30s to ensure the recording of task point passage.
Cylinder
Here you can define the radius around the task point, if TP Scheme Cylinder is selected.
Sector
Here you can define a Sector around the task point, if TP Scheme Sector is selected. Three items can be used for the sector configuration. The inner radius is a circle around the task point. It can vary between zero and the outer radius. If inner and outer radius are equal you will get a full circle as result without any sector. The angle of the sector can vary between 90 and 180 degrees. For a so called FAI photo sector inner radius is zero, outer radius infinite and the angle 90 degrees.
Shape
Here you can define, if the used Cylinder/Sector shall have a outer border line and shall be filled with a transparent color. Switching off both will make the task point invisible.
