Preflight Settings

Common

Here you can enter different common preflight data.

Arrival Altitude
The arrival altitude is your preferred altitude above terrain when on final glide. It is used for the calculation of the glide path value. The value entered here is added to the elevation of the selected target. The arrival altitude is given in meters, if altitude unit meters is selected. Otherwise foot is used. For most situations, the default 200 meters would be sensible.
QNH
The QHN must be entered in hPa (Hecto Pascal). It is only considered in the warning distance calculation to standard pressure airspaces. The default value is 1013hPa.
The Cumulus IGC logger does not fulfill the IGC requirements in all points. There is no security G-Record written into the log file because the PDA device cannot be secured against external manipulations. Furthermore there are not always pressure altitude values available as required by the IGC. For more information about the FAI IGC requirements see here.
Autostart logging
If you activate this check box, Cumulus starts automatically the IGC logging, when you be in motion faster as 5Km/h. The current logger status is signaled by different colors and to see in the status bar.
B-Record Interval
Interval for logging of the IGC B-Record. Default time is 10 seconds. Note, that lower times will increase the log file size.
K-Record Interval
Interval for logging of the IGC K-Record. Default is None, that means switched off. If you enter a valid time, the following additional items are logged, if they are available. Otherwise default values are used.

  • HDT, true heading as 3 numbers
  • TAS, true airspeed as 3 numbers with unit kph
  • WDI, wind direction as 3 numbers
  • WSP, wind speed as 3 numbers in kph
  • VAT, variometer speed in meters with sign +/-, 3 numbers with 3 decimal numbers. If the speed is zero the sign is reset to space.

Note, the activation of this option will increase the log file size.
