Preflight Settings

Common

Here you can enter different common preflight data.

Arrival Altitude
The arrival altitude is your preferred altitude above terrain when on final glide. It is used for the calculation of the glide path value. The value entered here is added to the elevation of the selected target. The arrival altitude is given in meters, if altitude unit meters is selected. Otherwise foot is used. For most situations, the default 200 meters would be sensible.
QNH
The QHN must be entered in hPa (Hecto Pascal). It is only considered in the warning distance calculation to standrad pressure airspaces. The default value is 1013hPa.
Taskpoint switch radius
If you have activated a flight task, cumulus supports a automatically switch to the next taskpoint, when a certain distance is fallen short to the current taskpoint. Here you can define this radius around the taskpoint. The switch will be executed, if you have approached the sector described by the radius and reached the minimum distance to the selected taskpoint. The radius is adjustable between 0.1...10.0 in your selected distance unit (Kilometer, Miles or nautical Miles).
Autostart logging
If you activate this check box, cumulus starts automatically the IGC logging, when you start with moving.
