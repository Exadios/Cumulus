Settings - Airfields

Airfields

Cumulus can use two different sources openAIP and Welt2000 as airfield database. The default is Welt2000, which is also automatically downloaded, if it is missing or was updated. Welt2000 has stored all data in one file. openAIP uses one file per country. The advantage of openAIP is, that you can bring in directly your own contributions or corrections on their webpages at www.openaip.net. The disadvantage is, you have to download all the needed airfield files by hand because you need an account to login there and you have to accept their license policy.

Source
In the source box you can select the airfield source (openAIP / Welt2000) to be used. There are different forms provided for every source. In the headline of the form you can see the link to the appropriate webpage.

Welt2000 - Country Filter
Here you can define the countries to be read from the Welt2000 file. Several countries can be entered as a list separated by commas or spaces. The country identifiers have to be coded according to ISO-3166. If you define countries here, you do not need to do the same in the configuration file of welt2000.conf. The definitions from this GUI page will override the FILTER rule in the configuration file.

Welt2000 - Radius Filter
As an alternative to using the country filter, you can define a radius around your home position. All airfields lying inside this radius will be read from the Welt2000 data file. This option is only available when no country filter value is given. If the radius is zero, then a default radius of 500 kilometres is assumed.

Please Note! If the home position is too far away from the area defined by the filter or radius values, no airfield data are shown on the map.

Welt2000 - Load Outlandings
If the check box Load Outlandings is activated, all outlanding sites contained in the selected region will be loaded too.

Welt2000 - Install
Cumulus can download the Welt2000 file containing the airfield data from the relevant Internet web page.
Note, if Cumulus is in motion no download is carried out.

Welt2000 Data Reload
After a download or a change in home position or configuration, Cumulus will trigger an automatic reload of the Welt2000 data.

openAIP - Radius Filter
All airfields lying inside your home position radius will be read from the openAIP data files, which are selected on the load page.

Please Note! If the home position is too far away from the area defined by the radius value, no airfield data are shown on the map.

openAIP - Load
Pressing the Load button opens a table showing the loadable openAIP airfield files. If Select all is checked, all files are loaded by Cumulus. If you want not to load all, deselect as first the checkbox Select all. Only then you can make single selections of airfield files. If you select a whole table row, you can remove the selected data file by pressing the trash button. In the headline of the table you can see the file directory.
In the List Display area you can define some general list properties applied to airfield, waypoint, reachable and other lists.
