Settings

Airfields

Cumulus uses the Welt2000 file as airfield and outlanding data base. See here for more information.

In the Welt2000 area you can set different options for airfield data loading.

Welt2000 Country Filter
Here you can define the countries to be read from the Welt2000 file. Several countries can be entered as list separated by commas or spaces. The country identifiers have to be coded according to ISO-3166. If you define countries here, you should not do the same in the configuration file of welt2000.conf. The definitions from the GUI surface will overwrite the FLITER rule in the configuration file.
Welt2000 Radius Filter
As alternative to the country filter you can define a radius around your home position. All airfields laying inside this radius will be read from the Welt2000 data file. This option is only available, when no country filter has been setup. If the radius is zero, then internally a radius of 500 Kilometers is assumed.
Please Note! If the home position is too far away from the area where you do fly, no airfield data are to see on the map.
Welt2000 Load Outlandings
If the check box Load Outlandings is activated all outlandings contained in the selected region will be loaded too.
Proxy : Port
Define here a proxy if it is needed for downloads from the Internet. The syntax expects the following format. Proxy name separated by a colon from the port.
Install Airfields and Welt2000 File Name
Cumulus can download the Welt2000 file which contains the airfield data from the regarding web page out the Internet. Important for that is the right file name. Unfortunately the name is combined with a date and can be changed from time to time. Please check here the correct spelling, if the download fails. Look at the web page into the last row of the table where the entry WELT2000.TXT is to find. In the right column you can see a date link. Go onto this link and check if the last component do match the input of the editor field.
If Cumulus is in move (faster as 5 Km/h) no download is executed. If the download request is processed you will get once a result message. Cumulus does not record a special log file for that. If Cumulus is started in a X-Terminal window you can see the single result as debug message.
Welt2000 Data Reload
After a download, a home position or configuration change Cumulus will trigger an automatic reload of the Welt2000 data.
In the list display area you can define some general properties of the airfield, waypoint, emergency lists.
