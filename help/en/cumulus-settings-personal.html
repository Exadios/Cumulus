Settings

Personal

Here you can enter your personal data. This data is mostly used for the igc logfiles. The homesite coordinates are used for manual navigation. The homesite coordinates can also be set via the airfield and waypoint pages. In this case the data are overtaken from these items and a manuell entry is superfluous.

You can select one of the additional provided languages for cumulus. The default language is english.

With the Button Data Directory and the text entry field beside it you can define a new storage location for your waypoint, task and IGC logger files. The default storage area of Cumulus is $HOME/cumulus. On Maemo (Nokia Internet Tables) it is recommend to define a new location, which lays on one of the MMC cards (/media/mmc1 or /media/mmc2). The MMC cards are accessable with the Maemo file manager and also via USB. So you can transfer yours data files between your PC and the Internet Tablet very conveniently.

In the text field you can enter a new directory loaction laying on your MMC card. When you press the Data Directory button, a directory dialog is opened and you can navigate to your new storage directory area. The directory dialog shows only existing directories. If you want to create a new subdirectory you have to add the missing pathes in the text field by hand. Cumulus tries then to create the new subdirectory, if you leave the settings dialog with the Ok button.

E.g.

/media/mmc1/cumulus or /media/mmc2/cumulus

should be good locations.

Note, if you change the data directory no files will be transfered automatically by Cumulus to the new area. Such moves must be done alone by the user.
