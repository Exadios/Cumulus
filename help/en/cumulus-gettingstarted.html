Getting Started

To take full advantage of Cumulus, you need a GPS receiver and a set of map files.
PDA
As long as the program runs on the PDA and you are in flight, the automatic shutdown of the screen to save battery life is deactivated. For operation over several hours you need to supply external power to your PDA.
GPS
You can use any GPS receiver or a bluetooth mouse that can be hooked-up to your PDA and outputs NMEA data records, either by using a serial or a blue tooth connector. Cumulus can be used without a GPS receiver, but is far more usefull with such a device.
Once you manage to connect your GPS to Cumulus, select the correct device and baud rate in the settings dialog. Refer to your GPS manual for the correct settings. For most serial devices, device /dev/ttyS0 will do, for bluetooth /dev/rfcomm is most common, for N810 use /dev/pts/0. The default NMEA connection speed is 4800 bps for bluetooth we recommend to use 115200 bps,

Press G to check the GPS status. Use the Escape key to leaf the status page.
Maps
Cumulus uses the same maps as KFLog, the KDE Flight Logger. Download the maps you need and install them on your PDA or on a MultiMediaCard or SecureDigital Card. See maps for more details on downloading and installing maps.
Waypoints
Cumulus uses waypoints to facilitate navigation. You can either edit waypoint files on the PDA or prepare them on the desktop using KFLog and transfer them to the PDA. See waypoints for more details on waypoints.
Tasks
Cumulus supports also the definition of flight tasks. You can either edit the task file on the PDA or prepare them on the desktop using KFLog and transfer them to the PDA. See tasks for more details on tasks.
