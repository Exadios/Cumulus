Settings

Airspaces

The list shows you which kind of airspaces can be drawn onto the map. Check the items of the airspaces to be displayed. To change the color of the airspace border or the airspace area touch the related field in the list. That will opens a color dialog, where you can change the current color. Pressing button Color Defaults will assign the programmed default colors to all airspaces.

Note that Cumulus will only warn you for airspaces that are drawn.

You will get optical and acoustical warnings if you are near, very near or inside of specified airspaces. If you press the button Airspace Warnings you can set lateral and vertical distances (separately for above and below) that will be applied when generating warnings. If the distance to an airspace is smaller than the specified value, the position is considered near respectively very near to the airspace.

If you press the button Airspace filling you can enable, that the airspaces are filled with a transparent color with different transparency according to the near of the airspace. If you disable the airspace filling, a fixed pattern is used for filling.

The units used for the distances can be configured via the Units page. The corresponding item is Altitude. If unit meters is selected, meter will be used as distance otherwise food.
