Preflight Settings - Flarm

Flarm

Here you have the possibility to readout and set several Flarm data. On this page you can upload a selected flight task together with its related data to the Flarm device.

Window description
In the upper window part are displayed in the Flarm Data panel different read only items from the Flarm device. If the fields are filled with ??? no data could be read from the Flarm device. In this case there seems to be no active connection available. The abbreviated labels have the following meaning:
  • SN: Serial number
  • RId: Radio Identifier
  • Sv: Severity of issue
  • Err: Error code
  • ODB: Obstacle data base version. Empty, if no data base is installed.
  • HW: Hardware version
  • Sw: Software version
  • Igc: IGC version. Empty on none approved IGC Flarms.
Note! If Flarm reports an error code in the framed box beside the Err: label, you can click on that error code and will get a textual description of the error code.
LogInt
Here you can define the log interval of the Flarm IGC logger. The possible setting range is 1...8 seconds. The default is set to two seconds.

Priv
Here you can set/unset the so called Stealth mode of the Flarm device. 0 means off and 1 means on. If you don't know, what it is, let it switched off (0)! That is better for your personal safety.

NoTrk
Flarm opt-out option for ground tracking (“no track”). If set, the receiving ground stations should not process or store position data of your glider. 0 means off and 1 means on.

Task
In the displayed combo box at the right side of the task label you can select a task to be flown. Preselected is the currently activated task. But you can change here the current task selection, if you want to do that. The new selected task will be considered as the task to be used for flying. A task can also be selected at the preflight task page.

Read
Pressing this button will read out the set IGC data in the Flarm device and put them in the input fields of the window. Note, that a set IGC Task cannot be read out. That is not supported by Flarm.

Set
Pressing this button sets the Cumulus data in the input fields of the window.

Clear
Pressing this button will clear all input fields.

Write
Pressing this button will upload all input field data to the Flarm device inclusive a selected or empty task.

Close
Pressing this button will close the window.

Please note the following:

  • Task support works only for the IGC approved Flarm devices!
  • To get activated an uploaded task, you have to cycle the power switch of the Flarm device.
