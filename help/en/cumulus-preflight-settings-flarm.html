Preflight Settings - Flarm

Flarm

Here you have the possibility to readout and set Flarm data. On this page you can upload a selected task together with other related task data to the Flarm device.

Window description
In the upper window part are displayed in the Flarm Data panel different read only items from the Flarm device. If the fields are filled with ??? no data could be read from the Flarm device. In this case seems to be no active connection available. If Flarm reports an error code in the frame behind the Err: label, you can click on that error code and will get an textual description of it.
LogInt
Here you can define the log interval of the Flarm IGC logger. The possible setting range is 1...8 seconds. Flarm default are two seconds.

Priv
Here you can set/unset the so called Stealth mode of the Flarm device. 0 means off and 1 means on. If you don't know, what it is, let it switched off (0)! That is better for your personal safety.

NoTrk
Flarm opt-out option for ground tracking (“no track”). If set, receiving ground stations do not process or store position data of your glider. 0 means off and 1 means on.

Read
Pressing this button will read out the set IGC data in the Flarm device and enter them in the input fields of the window.

Set
Pressing this button sets the Cumulus data in the input fields.

Clear
Pressing this button will clear all input fields.

Write
Pressing this button will download all input field data to the Flarm device inclusive a selected or empty task.

Close
Pressing this button will close the window.

Please note, task support works only for IGC approved Flarm devices!

The task to be downloaded to the Flarm device has to be selected on the task page. If the task definition is empty, an empty task is transfered to the Flarm device. Note! To get running a transfered task, you have to cycle the power switch of the Flarm device.
